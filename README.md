# Django Models

## Reference:

- https://docs.djangoproject.com/en/5.1/topics/db/models/

## venv

```sh
python -m venv .venv
source .venv/bin/activate
```

## Install packages

```sh
python -m pip install -r requirements.lock.txt
```

## Run migrations

```sh
python manage.py migrate
```

## Run server

```sh
python manage.py runserver
```
